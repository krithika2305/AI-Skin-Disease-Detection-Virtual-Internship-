# 🧠 AI Skin Disease Detection System

A full-stack AI-powered web application that detects and classifies skin diseases from uploaded images using deep learning. The system helps users identify potential skin conditions quickly and encourages early medical consultation.

## 🚀 Features
📸 Upload skin images for analysis
🤖 Deep Learning-based disease prediction
🩺 Detects multiple skin diseases
📊 Displays prediction results with confidence score
🌐 User-friendly web interface
⚡ Fast and accurate image processing

## 🛠️ Tech Stack
### Frontend:
HTML
CSS
JavaScript
### Backend:
Python (Flask)
Machine Learning:
TensorFlow
Keras
CNN (Convolutional Neural Network)
### ⚙️ How It Works
User uploads a skin image
Image is preprocessed (resizing, normalization)
CNN model extracts features
Model predicts the skin disease
Result is displayed with confidence score
### 🧠 Machine Learning Details
Algorithm: Convolutional Neural Network (CNN)
Image-based classification model
Trained on labeled skin disease dataset
Includes image preprocessing and normalization
Outputs predicted class with probability score

### ⚠️ Note
Model accuracy depends on dataset quality
This system is for educational purposes only
Not a replacement for professional medical advice
